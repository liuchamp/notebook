Sets 相关命令

| 命令 | 作用 | 示例 |
| :--- | :--- | :--- |
| SADD | 添加一个或者多个到redis中 |  |
| SCARD | 返回key中元素个数 |  |
| SDIFF | 返回由第一个集合和所有连续集合之间的差得出的集合成员。 |  |
| SDIFFSTORE | 此命令等于SDIFF，但不返回结果集，而是存储在目标位置。如果目的地已经存在，它将被覆盖。 |  |
| SINTER | 返回所有给定集合的交集所得的集合成员。 |  |
| SINTERSTORE | 此命令等于SINTER，但不返回结果集，而是存储在目标位置。如果目的地已经存在，它将被覆盖。 |  |
|  |  |  |
